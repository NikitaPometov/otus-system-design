=== Потоки данных и высокоуровневое взаимодействие

[upperroman]
. Просмотр каталога
[arabic]
.. Клиент -> API Gateway -> Сервис Товаров
.. Сервис Товаров читает данные из своей БД и возвращает список товаров

. Работа с корзиной
[arabic]
.. Клиент -> API Gateway -> Сервис Корзины
.. Сервис Корзины синхронно запрашивает данные о товарах в Сервисе Товаров
.. Сервис Корзины хранит содержимое корзины в кратковременном хранилище с TTL

. Оформление заказа
[arabic]
.. Клиент -> API Gateway -> Сервис Заказов
.. Сервис Заказов синхронно запрашивает состав корзины в Сервисе Корзины
.. Сервис Заказов синхронно валидирует адрес в Сервисе Проверки Адресов -> внешняя система проверки адресов
.. Сервис Заказов синхронно проводит оплату через Сервис Платежей -> внешняя система платежей
.. Сервис Заказов сохраняет заказ в своей БД
.. Сервис Заказов публикует события в топики Сервиса Уведомлений и Сервиса Товаров

. Уведомления
[arabic]
.. Сервис Уведомлений читает события из топиков
.. Сервис Уведомлений отправляет сообщения через внешние системы уведомлений

. Управление пользователями
[arabic]
.. Администратор -> API Gateway -> Сервис Управления Пользователями
.. Сервис Управления Пользователями изменяет профиль/статус пользователя в своей БД
.. Сервис Управления Пользователями публикует событие для Сервиса Уведомлений

. Управление каталогом
[arabic]
.. Супервайзер -> API Gateway -> Сервис Товаров
.. Сервис Товаров изменяет данные товаров в своей БД
